SilcConfigFile

SYNOPSIS

    typedef struct SilcConfigFileObject SilcConfigFile;

DESCRIPTION

A File object holds the data contained in a previously loaded file by the silc_config_open() function. This is an internally allocated struct and must be used only with the helper functions.
