silc_xml_parse

SYNOPSIS

    SilcBool silc_xml_parse(SilcXMLParser parser,
                            const unsigned char *data,
                            SilcUInt32 data_len);

DESCRIPTION

Parse XML data `data' of length of `data_len' bytes. Returns TRUE after the data has been parsed. The handler callback functions set for `parser' will be called while parsing the XML data.

Returns FALSE and set silc_errno and silc_errno_reason if error occurs.
