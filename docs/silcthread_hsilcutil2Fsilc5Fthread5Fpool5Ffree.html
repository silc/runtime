silc_thread_pool_free

SYNOPSIS

    void silc_thread_pool_free(SilcThreadPool tp, SilcBool wait_unfinished);

DESCRIPTION

Free the thread pool. If `wait_unfinished' is TRUE this will block and waits that all remaining active threads finish before freeing the pool.
