silc_file_read

SYNOPSIS

    int silc_file_read(int fd, unsigned char *buf, SilcUInt32 buf_len);

DESCRIPTION

Reads data from file descriptor `fd' to `buf'. Returns the amount of bytes read, 0 on EOF or -1 on error and sets silc_errno.
