silc_bit_clear

SYNOPSIS

    SilcBool silc_bit_clear(volatile unsigned long *bitmap,
                           SilcUInt32 bitmap_size,  SilcUInt32 bit);

DESCRIPTION

Clear bit number `bit' in the `bitmap' of size of `bitmap_size'. Returns FALSE on error and sets silc_errno.
