silc_async_halt

SYNOPSIS

    SilcBool silc_async_halt(SilcAsyncOperation op);

DESCRIPTION

Halt the execution of the asynchronous operation. If the operation supports this feature, it is guaranteed that the operation is halted and its execution is not progressed until the silc_async_resume function is called. The operation still can be aborted even if it is halted. If this function is not supported, calling this has no effect and the function returns FALSE. This function is for the upper layer that controls the asynchronous operation.
