silc_buffer_stream_send

SYNOPSIS

    SilcBool silc_buffer_stream_send(SilcStream stream,
                                     SilcBuffer buffer);

DESCRIPTION

Sends `buffer' to the buffer stream indicated by `stream'. If the `stream' is not a buffer stream created by silc_buffer_stream_create this will return FALSE. Returns FALSE on error and sets silc_errno.
