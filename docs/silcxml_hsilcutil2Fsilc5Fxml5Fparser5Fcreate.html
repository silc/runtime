silc_xml_parser_create

SYNOPSIS

    SilcXMLParser silc_xml_parser_create(SilcXMLParams params,
                                         SilcXMLParserHandler handler,
                                         void *context);

DESCRIPTION

Create XML parser and return in. The `handler' contains the callback functions to be called while parsing XML data. The `context' is delivered to each callback function. The `params' define parser parameters, and may be NULL. The parser parses XML data with UTF-8 encoding. All characters delivered to callbacks are in UTF-8 encoding.
