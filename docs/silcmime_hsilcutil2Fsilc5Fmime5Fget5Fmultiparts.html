silc_mime_get_multiparts

SYNOPSIS

    SilcDList silc_mime_get_multiparts(SilcMime mime, const char **type);

DESCRIPTION

Returns list of the parts from the MIME message `mime'. Each entry in the list is SilcMime context. The caller must not free the returned list or the SilcMime contexts in the list. Returns NULL if no parts exists in the MIME message. Returns the multipart type (like "mixed") into `type' pointer.
