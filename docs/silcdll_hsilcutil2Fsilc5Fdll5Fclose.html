silc_dll_close

SYNOPSIS

    void silc_dll_close(SilcDll dll);

DESCRIPTION

Closes the shared object indicated by `dll'. Any symbol retrieved from the `dll' with silc_dll_getsym will become invalid and cannot be used anymore.
