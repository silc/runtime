silc_dlist_sinit

SYNOPSIS

    static inline
    SilcDList silc_dlist_sinit(SilcStack stack);

DESCRIPTION

Initializes SilcDList. Returns the SilcDList context or NULL on error. This is same as silc_dlist_init but allocates the memory from `stack' if `stack' is non-NULL.
