SILC_GETOPT_INIT

NAME

    #define SILC_GETOPT_INIT ...

DESCRIPTION

Macro used to initialize SilcGetOptStruct before calling silc_getopt.

EXAMPLE

    SilcGetOptStruct op = SILC_GETOPT_INIT;
