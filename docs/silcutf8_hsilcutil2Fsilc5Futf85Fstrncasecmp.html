silc_utf8_strncasecmp

SYNOPSIS

    SilcBool silc_utf8_strcasecmp(const char *s1, const char *s2,
                                  SilcUInt32 n);

DESCRIPTION

The silc_utf8_strcasecmp() function compares the two strings s1 and s2, ignoring the case of the characters. It returns TRUE if the strings match and FALSE if they differ.

This functions expects NULL terminated UTF-8 strings. The strings will be casefolded and normalized before comparing. Certain special Unicode characters will be ignored when comparing.
